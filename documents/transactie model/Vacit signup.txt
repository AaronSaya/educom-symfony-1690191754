title VACIT Sign Up

actor User
participant Homepage
participant Signup page
participant Back End

database Database

==Transactie: Sign Up==
User->Homepage: Click sign up button
Homepage->Signup page: Open signup page
Signup page->User: Display signup page
User->Signup page: Enter username
Signup page->Back End: Check username
alt username does not exist
Back End->Signup page: username does not exist 
else username does exist
Back End->Signup page: username does exist 
Signup page->User: Warning message username already exist
end
User->Signup page: Enter password
Signup page->Back End: Check password
opt password requirements not met
Back End->Signup page: Requirements not met
Signup page->User: Message requirements not met
end
User->Signup page: Enter password confirmation
Signup page->Back End: Check confirmation
opt password confirmation
Back End->Signup page: Confirmation failed
Signup page->User: Message confirmation failed
end
User->Signup page: Confirm signup
alt Signup succes
Signup page->Back End: Send data
Back End->Database: Save data
Database->Back End: Successfull signup
Back End->Signup page:Successfull signup
Signup page->User: Redirect successfull signup page
else Error signup failed
Back End->Signup page: Signup failed
Signup page->User: Message signup failed
end
